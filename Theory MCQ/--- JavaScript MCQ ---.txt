--- JavaScript MCQ ---

1. Which of the following is a correct data type in JavaScript?

a) Integer
b) Float
c) Undefined
d) Character
Answer: c) Undefined

2. Which of the following methods can be used to display data in JavaScript?
a) document.write()
b) console.log()
c) alert()
d) All of the above
Answer: d) All of the above

3. What is the correct syntax for referring to an external script called "script.js"?
a) <script name="script.js">
b) <script href="script.js">
c) <script src="script.js">
d) <script link="script.js">
Answer: c) <script src="script.js">

4. Which of the following is a correct way to create a function in JavaScript?
a) function = myFunction()
b) function myFunction()
c) myFunction()
d) None of the above
Answer: b) function myFunction()

5. How do you call a function named "myFunction" in JavaScript?
a) myFunction.call()
b) call myFunction()
c) myFunction()
d) None of the above
Answer: c) myFunction()

6. Which of the following is used to declare a variable in JavaScript?
a) let
b) var
c) const
d) All of the above
Answer: d) All of the above

7. What is the output of the following code: console.log(2 + '2');?
a) 22
b) 4
c) '4'
d) '2 2'
Answer: a) 22

8. Which of the following is the correct way to declare a constant in JavaScript?
a) var CONSTANT = 5;
b) constant CONSTANT = 5;
c) let CONSTANT = 5;
d) const CONSTANT = 5;
Answer: d) const CONSTANT = 5;

a) true
b) false
c) NaN
d) Undefined
Answer: b) false

a) "number"
b) "NaN"

11. Which of the following is not a loop in JavaScript?

b) while
c) repeat
d) do...while
Answer: c) repeat

12. Which of the following statement is used to exit from a loop?
a) break
b) exit
c) stop
d) return
Answer: a) break

13. What does the continue statement do in JavaScript?
a) It terminates the loop.
b) It skips the current iteration of the loop.
c) It jumps out of the loop.
d) None of the above
Answer: b) It skips the current iteration of the loop.

14. Which of the following is used to check equality in JavaScript (with type checking)?
a) =
b) ==
c) ===
d) !=
Answer: c) ===

Functions and Scope
16. In JavaScript, what is the scope of a variable declared with var inside a function?
a) Block scope
b) Function scope
c) Global scope
d) Local scope
Answer: b) Function scope

17. What is a higher-order function in JavaScript?
a) A function that takes another function as an argument or returns a function
b) A function that is written in uppercase
c) A function that has more than one parameter
d) None of the above
Answer: a) A function that takes another function as an argument or returns a function

18. Which of the following is an example of an Immediately Invoked Function Expression (IIFE)?
a) function() { console.log('IIFE'); }
b) (function() { console.log('IIFE'); })();
c) var IIFE = function() { console.log('IIFE'); };
d) None of the above
Answer: b) (function() { console.log('IIFE'); })();

19. What will be the output of this code: var x = 1; function print() { var x = 2; console.log(x); } print(); console.log(x);?
a) 2 1
b) 1 2
c) 2 2
d) 1 1
Answer: a) 2 1

20. What is the default return value of a function in JavaScript if no return statement is provided?
a) null

Answer: c) `Hello, ${name}`

30. What does the bind() method do in JavaScript?

b) catch { ... } try { ... }

Answer: a) try { ... } catch { ... }

d) e is not defined
Answer: a) Oops

43. Which statement is used to explicitly throw an exception in JavaScript?
a) throw
b) catch
c) error
d) try
Answer: a) throw

44. What is the purpose of the finally block in JavaScript?
a) To always execute after the try and catch blocks, regardless of whether an exception occurred

Answer: a) To always execute after the try and catch blocks
